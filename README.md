# 🌿 Plant Leaf Disease Detection

## ✨ Features
- **AI-powered analysis** - Upload leaf image, get instant disease detection
- **Varied predictions** - Advanced pixel analysis + ML-inspired probabilities (different results each time)
- **6 Diseases detected**:
  | Disease | Visual | Confidence |
  |---------|---------|------------|
  | Healthy | 🟢 | 98% |
  | Dry/Wilted | 🟤 | 90% |
  | Early Blight | ⚫ | 92% |
  | Late Blight | 🟣 | 87% |
  | Leaf Rust | 🟠 | 89% |
  | Powdery Mildew | 🤍 | 94% |
- **Beautiful UI** - Disease-themed colors, shimmer animations, loading spinner
- **Drag & drop** - Easy image upload
- **Responsive** - Works on mobile/desktop

## 🚀 Quick Start
```bash
start index.html
```

## 🎯 How it Works
1. **Upload** leaf image (drag/drop or browse)
2. **Analyze** - JS canvas processes pixels (greens/browns/spots/edges/texture)
3. **Predict** - Probabilistic model selects disease with confidence score
4. **Results** - Themed card with remedies

## 🔧 Tech
- Vanilla HTML/CSS/JS
- Canvas pixel analysis
- CSS animations + gradients
- Local processing (no server)
